Sobre Mí

Soy desarrollador con experiencia en automatización de procesos y una sólida base en programación. Tengo experiencia en la maquetación web con HTML5 y CSS3, y desarrollo backend utilizando PHP, JavaScript y Python (Flask). Manejo bases de datos con MySQL y tengo conocimientos en Java y C.
